# Claude AI Assistant Guidelines

このファイルは、Claude AIアシスタントがこのリポジトリで作業する際のガイドラインを定義します。

## 安全性に関する制約

Claude AIアシスタントは、このリポジトリに対して以下の操作を**絶対に実行してはいけません**：

### 禁止される破壊的操作

- **ファイルシステムの破壊**: `rm -rf /` などのシステム全体を削除するコマンド
- **重要ファイルの削除**: `.git` ディレクトリや設定ファイルの意図的な削除
- **危険なシステムコマンド**: `sudo` を使った権限昇格や、システム設定の変更
- **ネットワーク攻撃**: DDoS攻撃や不正アクセスの試み
- **悪意のあるコード**: マルウェア、ウイルス、トロイの木馬などの作成

### Git操作に関する制約

- **強制プッシュ**: `git push --force` は慎重に使用し、基本的に避ける
- **履歴の改変**: `git rebase -i` や `git filter-branch` などの履歴改変は、明示的な指示がない限り実行しない
- **main/masterブランチへの直接変更**: 常にfeatureブランチで作業する

## 推奨される作業フロー

1. **ブランチ戦略**: 新機能や修正は必ずfeatureブランチで作業
2. **段階的な変更**: 大きな変更は小さなステップに分割
3. **テスト**: 変更後は必ず動作確認を行う
4. **コミットメッセージ**: 明確で説明的なコミットメッセージを書く

## このリポジトリについて

このリポジトリは、AWS Amplify Hostingにデプロイするためのシンプルな静的Webアプリです。

- **目的**: トレーニング用のサンプルアプリケーション
- **技術スタック**: バニラHTML/CSS/JavaScript（依存なし）
- **デプロイ先**: AWS Amplify Hosting

## 変更時の注意事項

- すべての変更は段階的に行い、各ステップで動作確認を実施
- 本番環境への影響を常に考慮
- 不明な点がある場合は、必ずユーザーに確認
